O projeto tem como objetivo desenvolver uma aplicação web que permita a grupos de amigos que **jogam futebol organizarem os seus jogos e criarem um sistema "friendly" de ranking competitivo**. O ranking funcionará a nível interno (entre os jogadores de cada grupo). Cada utilizador poderá criar grupos ou integrar grupos e gerir os seus membros. A aplicação será uma web app em formato de dashboard, permitindo visualizar estatísticas básicas como golos, vitórias, empates e derrotas, tanto a nível individual como coletivo.

# Objetivos do Projeto

- Facilitar a organização de jogos de futebol informais entre amigos.
- Criar uma vertente de gamificação através de rankings internos (grupo).
- Incentivar a prática desportiva e o espírito competitivo de forma simples e acessível.
- Desenvolver uma web app, aplicando tecnologias modernas de frontend e backend.

# Tecnologias a utilizar

- **Design:** Figma.
- **Frontend:** React, TailwindCSS, Zustand (gestão de estados).
- **Backend:** Node.js, Express, Prisma ORM.
- **Base de Dados:** PostgreSQL (ainda por confirmar).
